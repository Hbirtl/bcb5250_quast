This is a python program that can parse FASTA files and return a txt file containing:

  • the number of contigs and the total length of them
  • the length of the largest contig
  • N50 value -- determined by adding the contigs in order, from smallest to largest, until the sum of the contigs is equal to half of the total length. The last contig added is the N50 value.

When the program is run it will ask for the user to input a FASTA file.
